How to Use This Page

  • Each row in the table is a payment method you can manage.
  • To activate or deactivate a payment method, use the checkbox under the Status column.
  • You can edit the Payment Name, API Keys, or Webhook Keys by clicking in the boxes and typing your changes.
  • Where to find these details:
    • Payment Name: This is the name of the payment provider (e.g., Stripe, PayPal, etc.).
    • API Key: You get this from your payment provider's dashboard (look for API or Developer section).
    • Webhook Key: This is also provided by your payment provider, usually in the Webhooks or API section of their dashboard.
  • When you are done, click Save Changes to apply your updates.
  • If you want to undo your edits before saving, click Reset Changes.
